## **2個找工作的錯誤心態**

當沒有工作經驗，開始找工作時，首先最重要的事情就是**找到第一份相關工作**獲得經驗。

可能花了很多時間、學費培養技能，製作作品集，撰寫符合職缺的履歷，以為這樣可以找到合適的工作。其實不是...

大多會遇到的可能是：

1.  打開應徵的104求職網站，職缺條件要求的經驗、技能、背景有門檻，要能跨過去心中。\
    感覺「即使投了，也不會找我面試...」，接著就**不敢投履歷**，怕投了就成**砲灰**，然後陷入**怕找不到工作的恐懼**。\
2.  期望在**沒有相關經驗、背景**條件下的**第一份工作**，可以找到公司背景優秀的**好工作**。例如：工作壓力小、待遇好、離家進，環境友善。

這些筆者也曾經有過，接下來會分享，2個常見的錯誤心態，在大學畢業後也曾經犯過其中ㄧ種。\
並分享「**找工作如何不將就現實條件**」，希望能幫助你打破心魔、不浪費寶貴的時間在選擇上。

### **怕找不到工作，改變原本設定的工作條件**

資管系畢業的我，想嘗試軟體工程師的工作，對於剛畢業的大學生、沒有實習經驗、寫程式也沒有特別出色，唯一與程式有關的經歷是「畢業專題」。

因為資訊人才需求每年提升，看似很多選擇，其實對於**沒經驗的大學生**能選擇的**有限**，有哪個公司願意花錢「培育人才」？大部分公司要找的人才，有三種：

1.  能技術引進
2.  能幫公司省錢
3.  能立即上手的即戰力

**新手上路**唯一的優點是年輕、白紙一張好培訓，但願意的通常是「規模較大的企業、或資本雄厚的新創公司」，當時大台北地區發展「軟體產業」最興盛的就屬「內湖科技園區」，「機會多、選擇多」。可是，住新北地區的我，車程「來回3小時」，大概可以從台北到台中了(笑)。

當時給自己設定的**目標**是

「找到第一份軟體工程師，瞭解工程師的工作內容，至少待滿一年，如果真的做不來就換跑道。」

所以**即使是「內湖」的工作機會，也願意嘗試**。設定這樣的工作條件，**違背我期望能找到「通勤一小時」的工作**，但背後的原因是「擔心找不到工作」**，**深怕面臨「畢業即失業」、「畢業後還要靠家裡養」，雖然家裡也沒有給壓力，讓我自由發揮，不工作至少要有事情做就好。\
\
但多少會有「社會傳統觀念」的無形壓力，讓我想趕快找到工作。

### **職缺條件其中幾項不會，就不敢投，或覺得可疑**

不論有無工作經驗都會遇到，不知道**如何看職缺條件**，是不是與自己條件相符，看到職缺就**嚇到不敢投**，覺得投了一定獲得**無聲卡**，或是職缺需要的**技能**經驗值要很高，認為自己無法勝任。

另一種，看到職缺描述的反應是**覺得有詐**：條件要求很多，工作壓力會不會很大？要常加班等等，還沒有面試，就有很多「公司會很壓榨員工」的小劇場出現。

## **找工作，如何不將就「現實條件」**

工作幾年後，意識到只要將自身問題分析好，遇到困難時就不會卡住太久。

也就是將自己的優勢、劣勢的條件寫出來，看看求職市場中，你能把握的機會有哪些。

**以筆者大學剛畢業為例：**

| 優勢 | 劣勢 |
| --- | --- |
| 年輕、時間多 | 私立科技大學 |
| 本科系畢業 | 對業界所需技術不熟 |
| 有JAVA證照 | 沒有相關工作經驗 |
| 沒有正職工作經驗、白紙一張 |  |
| 肯配合、主動積極、適應力強 |  |
| 有在知名企業打工的經驗，雖然與工程師工作無關 |  |

**基於這些條件，想找什麼工作？**

| 工作條件 |
| --- |
| 公司願意培養技術人員 |
| 公司產業不拘，不要博奕類 |
| 通勤距離一小時可到 |
| 主管、同事要能溝通、好相處 |
| 公司待遇正常、要能付的薪水、有保勞健保 (看似很基本，還是有些小公司會佔員工便宜) |
| 我要待至少一年，有完整的正職相關經驗 |

**那會做的心裡準備是？**

| 假設情況 | 如何面對 |
| --- | --- |
| 如果三個月都沒有找到工作 | 重頭學習技術，到資策會學習 |
| 找到後，如何在公司生存至少一年 | 下班後持續學習技術 |

其實**最壞的情境是「找不到理想的」**，世界沒有崩壞，如果沒有找到也許是準備還不夠，也許是「時候未到」，但問題**最後都會被你解決的**。

從經驗來看，**如何不將就現實條件？**

就是**對於自己要的東西，有十足的自信，而且要持續行動，讓目標達成**。

也許一開始會徬徨、迷惘，都是正常的反應，常聽到「恐懼來自於無知」，想要消除恐懼，勢必要將一些你不知道的領域、問題，變成「喔～這個我大概知道」的程度。

我們不可能每件事都知道，沒有人第一份工作什麼都會知道，大家也都是從沒有工作經驗，變成有工作經驗。

當**持續行動的過程中，生活中會無形的增加許多「選擇」**，人生不是「二元思維」就能面對全部的問題，讓自己勇敢面對生活中的可能，就有機會發展出你意想不到的情境。

後來工作幾年後，職場上遇到**會面試，但不會做事**的也很多，在研究所畢業後規劃從「專案管理」轉職回「前端工程師」(開發網站，負責實現頁面功能的工作)的過程中，**大膽設定自己想要的工作條件**，在一個月內找到自己理想條件的工作，通勤距離終於在正常的40分鐘～1小時，公司環境也比以往好很多。

也許，因為我改變工作條件的第一份工作，現在的我，不會找到現在的工作，我想分享給沒有經驗的你：

**如果在「猶豫設定工作條件」**，真心建議就照你想要的條件去找，如果都沒有找到，就回頭調整、補足你為滿足的條件，邊找邊修正，到最後會發現你有很多選擇。

**如果在害怕「職缺描述需要技術經驗」而不敢投**，也真心建議你就用力投下去，有投有機會，投了才知道你想要去的地方是「長什麼樣子」你還需要補足哪些才能達到目標，因為害怕而不敢投，就沒有機會見識到你希望的樣子。

就像，很多我們看似沒有特殊之處的同學，莫名就和校花、校草在一起，問了才知道，原來大家以為校花、校草難相處都不敢告白，結果沒有特別出色的同學，勇敢的告白後，他們就在一起了。(這個例子舉的不是特好，但我相信你懂的。笑)

很多想去 Google, Amazon 科技大頭的工程師們，也是勇敢的投履歷，即使沒有錄取還是用力刷題目，做好十足的準備，面對下次能夠成功的機會。
